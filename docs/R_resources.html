General Resources

  • Posit cheat sheets - one-pagers for major packages. seriously, check them out.

  • RStudio keyboard shortcuts

    Some of my favorites are:

    • Ctrl + Enter: run current line or selected code
    • Ctrl + -: the assignment operator (<-)
    • Ctrl + Shift + c: comment all selected lines (insert # in front)
    • Ctrl + i: re-indent selected code
    • Ctrl + Alt + i: insert a code chunk into RMarkdown or Quarto file
    • F2: shows code used by function

  • Videos and Webinars from Posit - Several helpful webinars and presentations from past conferences.

Free Online R Books and Resources

Some more specialized than others

R and the tidyverse Generally

  • R For Data Science , 2nd edition - by Hadley Wickham, Mine Cetinkaya-Rundel, and Garrett Grolemund

  • Hands-On Programming with R - by Garrett Grolemund

  • Getting used to R, RStudio, and R Markdown - by Chester Ismay

  • ModernDive: Statistical Inference via Data Science - by Chester Ismay and Albert Y. Kim. Don’t let the title scare you; there’s a lot in here about R and RStudio generally. It is also a helpful intro to stats.

  • Advanced R , 2nd edition - by Hadley Wickham. You don’t have to be truly advanced to benefit from this book.

  • R Packages , 2nd edition - by Hadley Wickham and Jenny Bryan

Workflow Tips

  • “Workflow” Chapter of R4DS - by Wickham, Cetinkaya-Rundel, and Grolemund
  • “Projects” Chapter of ‘What They Forgot’ - from Jenny Bryan and Jim Hester. Talks specifically about working directories and Projects.
  • “Project-oriented workflow” - blog post by Jenny Bryan

Data Viz

  • Data Visualization: A Practical Introduction - by Kieran Healy. Also a good intro to the ggplot2 package; learn by doing.

  • Fundamentals of Data Visualization - by Claus O. Wilke

  • YouTube workshops from Thomas Lin Pederson (@thomasp85), a software engineer at Posit and co-author of the ggplot2 package (among other things). In 2020, he put on these impromptu workshops for beginner/intermediate R users:

    • Part One (~2.5 hours) is focused on the grammar of graphics and theory behind ggplot2
    • Part Two (~2 hours) discusses extension packages to the ggplot2 universe (gganimate and patchwork for example)
    • His materials are also available on GitHub

R Markdown

  • R Markdown: The Definitive Guide - by Yihui Xie, J.J. Allaire, and Garrett Grolemund

  • R Markdown Cookbook - by Yihui Xie, Christophe Dervieux, and Emily Riederer

Quarto

  • Quarto.org - the main website, with lots of tutorials and documention

  • Keynote Presentation: Hello, Quarto - given by Mine Centinkaya-Rundel and Julia Stewart Lowndes at Posit::Conf 2023. Fantastic overview of the many things you can do in Quarto.

  • Tutorial: Hello, Quarto - Web-based overview of Quarto

Geospatial Data

  • Geocomputation with R - by Robin Lovelace, Jakub Nowosad, and Jannes Muenchow. 2nd edition in progress.

  • Spatial Data Science - by Edzer Pebesma and Roger Bivand

Version Control (Git and Github)

  • Happy Git with R - online book by Jenny Bryan
  • GitHub tutorials - from Karl Broman’s blog

Statistics using R (books and resources)

General stats

  • Book: Learning Statistics with R - by Danielle Navarro. Aimed at psychology researchers but has great motivating examples of why we even use statistics in the first place. Includes lots of clearly explained, worked examples using R code.

  • Book: ModernDive: Statistical Inference via Data Science - by Chester Ismay and Albert Y. Kim.

Ecological, Environmental, and/or Water Resources Statistics

  • Workshop: Introduction to Multivariate Data Analysis using vegan - by Gavin Simpson. vegan is an extremely popular package for running community analyses for ecology. Link is to GitHub repository with materials; that GitHub page links to the recording on YouTube.

  • Workshop: Advanced Community Ecological Data Analysis using vegan - by Gavin Simpson. Link is to GitHub repository with materials; that GitHub page links to the recording on YouTube.

  • Tutorial: GAMs in R - by Noam Ross. Helpful interactive overview of Generalized Additive Models, which are becoming popular in ecology and water resources analyses.

  • Gavin Simpson’s blog has many helpful posts on GAMs

  • Ben Bolker has put together a fantastic FAQ about mixed models
